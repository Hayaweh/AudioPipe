
[FR]
Ajouter une musique à AudioPipe :
[1] Placer le fichier de la musique souhaitée dans le dossier \Music
[2] Placer le fichier XML de la musique souhaitée dans le dossier \XML

/!\ LES DEUX FICHIERS DOIVENT PORTER LE MÊME NOM (SANS L'EXTENSION) /!\

[ENG]
Add a music to AudioPipe:
[1] Put music file in the folder \Music
[2] Put xml file in the folder \XML

/!\ THE TWO FILES NEED TO HAVE THE SAME NAME (WITHOUT EXTENSION) /!\
